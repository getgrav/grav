# Welcome to My Test Page

This is a demonstration of how Grav CMS works.

## Key Features

- **Easy to use**: Markdown-based content
- **Fast**: File-based, no database needed
- **Flexible**: Powerful theming and plugin system

### How It Works

Grav stores all content as Markdown files with YAML frontmatter. This makes it:

1. Version control friendly
2. Easy to backup
3. Portable across systems
